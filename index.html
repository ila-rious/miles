Overview

When to use
  • Multiple independent selections
  • Filtering data in tables or menus
  • Agreeing to terms and conditions
  • Parent/child batch selections
When not to use

Overview

When to use
  • Switching a setting on or off immediately
  • Binary choices with instant effect
  • Enabling or disabling features or preferences
When not to use
  • Action requires a confirmation step → use a checkbox
  • Selecting from more than 2 options → use radio buttons
  • Part of a form that needs submission → use a checkbox
